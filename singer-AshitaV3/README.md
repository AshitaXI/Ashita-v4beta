# Singer (Ashita v3 ONLY) - No HUD

## Install
Copy folder `singer` to:
- `Ashita/addons/singer/`

Load:
- `/addon load singer`

## Commands
- `/singer on|off|toggle|status`
- `/singer now`
- `/singer marcato "Song"`
- `/singer songs "Song1" "Song2" ...`
- `/singer playlists`
- `/singer playlist <name>`
- `/singer delay <sec>`
- `/singer interval <sec>`
- `/singer target <me|t|...>`
- `/singer nitro on|off|toggle`
- `/singer ccsv  on|off|toggle`

Shortcut:
- `/sing ...`
